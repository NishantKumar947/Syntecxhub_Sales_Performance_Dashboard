# Syntecxhub Sales Performance Dashboard

## 📌 Project Overview
This project was developed as a part of the **Syntecxhub Data Analytics Internship Program**. The main objective is to analyze raw retail/e-commerce sales data, clean it using Python, and build an interactive business dashboard in Tableau to track and evaluate overall sales performance.

⚙️ Core Tasks Implemented
1. Data Import & Cleaning (Python)
Using Pandas and OS libraries, the following preprocessing steps were performed:

Duplicate Removal: Identified and successfully removed duplicate entries to maintain data integrity.

Null Value Treatment: Handled missing data by filling blank Region values with 'Unknown' and missing Profit fields with 0.00.

Feature Engineering: Extracted dynamic time features like Year, Quarter, and Month from the original Order_Date column to support deep trend analysis in Tableau.

2. Interactive Dashboard Creation (Tableau)
The finalized dataset was connected to Tableau to construct key business insights:

Core KPI Cards: Created highly visible, formatted cards tracking Total Revenue and Total Profit.

Trend Analysis: Developed dynamic line and bar charts exploring Monthly Sales Trends and Quarter-over-Quarter (QoQ) performance.

Product Evaluation: Built sorted horizontal bar charts to instantly pinpoint the company's Top-selling products vs. Low-performing items.

Segment Comparisons: Designed a detailed Region-wise & Category-wise Highlight Table (Heatmap) for matrix comparison.

Interactivity: Embedded action filters ("Use as Filter") making the charts completely interactive for stakeholders.
